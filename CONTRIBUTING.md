# 如何贡献提示词

感谢你来到小X宝社区 **Genie-X-PAL 提示词中心**！这里汇集帮助肿瘤/罕见病患者及家属的 AI 提示词。

## 🌱 最简单的贡献方式（我的第一个 PR）
1. 复制 [`PROMPT_TEMPLATE.md`](./PROMPT_TEMPLATE.md)。
2. 新建一个文件夹，名字用你的提示词主题（如 `就诊问题清单/`），里面放一个 `prompt.md`。
3. 照模板填好，提 PR。
4. 不会写？把 issue 里"你要做什么"贴进 Claude / ChatGPT，让它帮你生成提示词正文，再润色。

## 要求
- **原创**，不抄袭他人提示词。
- **通俗**，患者/家属看得懂，少用术语。
- 医疗类提示词附「以主治医生为准」的提示，不做确定性诊断结论。
- 一个 PR 对应一个提示词，方便 review。

## 目录约定
- 每个提示词 = 一个顶层文件夹（参考 `MD-Decision-Navigator/`）。
- 文件夹名用中文主题，简短（如 `就诊问题清单`、`基因报告解读`）。

## 临床风险提示
提示词可能影响患者决策。请在提示词中明确"仅供参考"。详见根 README 的「临床风险」一节。

## 需要帮助？
在对应 issue 下评论，或 @辅导员，社区会有人带你完成第一个 PR。
